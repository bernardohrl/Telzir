## Rodadando a Aplicação

#### 1. Instale NPM e Angular
```
sudo apt-get install npm
npm install -g @angular/cli
```

#### 2. Suba o ambiente de desenvolvimento
```
cd simulador-telezir
ng s
```

#### 3. Acesse http://localhost:4200/


<br><br>
